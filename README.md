# Flask App with GitHub Actions CI

This project demonstrates a simple Flask application integrated with a
CI pipeline using GitHub Actions.

##  Tech Stack
- Python
- Flask
- GitHub Actions (CI)

##  Features
- Automatic CI pipeline on every push
- Installs dependencies
- Runs Flask app checks
